# Netflix-Content-Visualization
# 🎬 Netflix Content Visualization

A complete Data Analytics project built using **Python, Pandas, Matplotlib, and Seaborn** to analyze the Netflix Movies and TV Shows dataset.

This project performs data cleaning, preprocessing, exploratory data analysis (EDA), and creates professional visualizations.

---

# Project Features

✔ Data Cleaning

✔ Missing Value Handling

✔ Exploratory Data Analysis

✔ Movie vs TV Show Analysis

✔ Rating Distribution

✔ Release Year Trend

✔ Country-wise Analysis

✔ Genre Analysis

✔ Duration Analysis

✔ Monthly Content Added

✔ Correlation Heatmap

✔ Dashboard Visualization

---

# Dataset

Netflix Movies and TV Shows

Columns include:

- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Listed In
- Description

# Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Netflix-Content-Visualization.git
```

Move inside project

```bash
cd Netflix-Content-Visualization
```

Install libraries

```bash
pip install -r requirements.txt
```

Run project

```bash
python src/main.py
```
